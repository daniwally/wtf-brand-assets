# WTF Agency Color Palette

## Required Files

Upload these color assets to maintain brand consistency:

- `wtf-palette.json` - Color codes in JSON format
- `color-swatches.png` - Visual color samples

## JSON Format Example

```json
{
  "primary": {
    "brand_black": "#000000",
    "brand_white": "#FFFFFF",
    "brand_accent": "#[TO_DEFINE]"
  },
  "secondary": {
    "dark_gray": "#[TO_DEFINE]",
    "light_gray": "#[TO_DEFINE]",
    "premium_gold": "#[TO_DEFINE]"
  },
  "editorial": {
    "vogue_pink": "#[TO_DEFINE]",
    "sophisticated_navy": "#[TO_DEFINE]"
  }
}
```

## Color Applications

**Primary Colors:** Logo, main brand elements  
**Secondary Colors:** Backgrounds, text, accents  
**Editorial Colors:** Campaign-specific, premium feel  

## Usage in Workflows

Colors will be automatically applied to:
- Midjourney/Legnext banner generation
- ComfyDeploy AI workflows  
- Presentation templates
- Social media assets
