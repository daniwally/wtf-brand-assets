# WTF Agency - Brand Assets Repository

Central repository para todos los assets de marca de WTF Agency. Usado por Oscar y otros agentes para mantener consistencia de marca en todas las generaciones automáticas.

## 📂 Estructura

### `/logos/`
Logotipos principales WTF Agency en diferentes formatos y variaciones
- `wtf-logo-main.png` - Logo principal (color)
- `wtf-logo-white.png` - Logo blanco para fondos oscuros
- `wtf-logo-black.png` - Logo negro para fondos claros
- `/variations/` - Variaciones adicionales y formatos especiales

### `/typography/`
Fuentes corporativas y especímenes tipográficos
- `Inter-Thin.woff2` - Inter Thin (WTF Agency standard)
- `Inter-Bold.woff2` - Inter Bold (WTF Agency standard)
- `font-specimens.png` - Muestras visuales de tipografía

### `/colors/`
Paletas de colores corporativas
- `wtf-palette.json` - Colores hex/RGB/CMYK
- `color-swatches.png` - Muestras visuales de paleta

### `/key-visuals/`
Imágenes de referencia y templates visuales
- `/hero-templates/` - Templates principales para campañas
- `/campaign-references/` - Referencias de campañas exitosas
- `/style-guides/` - Guías visuales de estilo

### `/templates/`
Templates de diseño para diferentes formatos
- Banner layouts, social media templates, presentaciones

### `/guidelines/`
Documentación de marca y guidelines
- `brand-dna.md` - ADN de marca WTF Agency
- `style-references.md` - Referencias de estilo (Droga5 + Vogue)

## 🎯 Uso por Agentes

Este repositorio es sincronizado automáticamente con:
- Oscar (workspace-oscar) - Para generación de contenido
- Otros agentes del ecosistema WTF Agency

## ⚡ Quick Start

1. Subir logo principal WTF
2. Subir Inter typography files  
3. Subir 2-3 key visuals representativos
4. Definir paleta de colores básica
