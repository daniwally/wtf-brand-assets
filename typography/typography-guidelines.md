# WTF Agency Typography Guidelines

## Primary Typeface: Inter

### Font Hierarchy

**Headlines & Titles**
- Font: Inter Bold
- Usage: Main headlines, campaign titles, brand statements
- Style: Sophisticated, editorial confidence

**Body Text & Descriptions**
- Font: Inter Thin
- Usage: Body copy, descriptions, supporting text  
- Style: Clean, readable, premium feel

**Supporting Text**
- Font: Inter Regular (fallback)
- Usage: Legal text, credits, secondary info

### Brand Voice in Typography

**✅ DO:**
- Combine Thin + Bold for editorial contrast
- Use generous white space (magazine-style)
- Maintain sophisticated hierarchy
- Keep text concise and impactful

**❌ AVOID:**
- Multiple font weights in same text block
- Cramped layouts or tight line spacing
- Overly decorative or script fonts
- Generic corporate typography

### Technical Specifications

**Headlines (Inter Bold)**
- Size: 24-72px depending on format
- Line height: 1.1-1.3
- Letter spacing: -0.02em (tight)

**Body (Inter Thin)**  
- Size: 14-18px depending on format
- Line height: 1.4-1.6
- Letter spacing: 0 (normal)

### Applications

- **Banner automation:** Automated text overlay on Midjourney generations
- **Social media:** Consistent typography across platforms
- **Presentations:** Professional client decks
- **Campaign assets:** All branded materials
